Flex-direction: The flex-direction is used to define the direction of flexible item. 
            The default axis is horizontal in flexbox, so the items flow into a row. 

            Syntax:
            // Stacking flex items column wise
            flex-direction: column;

            // Stacking flex items from bottom to top
            flex-direction: column-reverse;

            // Stacking flex items row wise
            flex-direction: row;

            // Stacking flex items from right to left
            flex-direction: row-reverse;
