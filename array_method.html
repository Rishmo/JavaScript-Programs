How to use forEach array method and its uses

forEach

We use forEach function to access the individual property.

map

Map function is similar to forEach function but the only difference is when map returns the value it will create an array of similar size

filter

It will return the value which is true according to the callback

Every

Returns true if every element of array gives true for some funtion. Else returns false.

Some

Returns true if some element of array gives true for some funtion. Else returns false.

Reduce

Reduces the array to a single value.
